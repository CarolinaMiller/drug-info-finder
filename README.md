# Drug Info Finder

Web app to search FDA-approved drug information using the OpenFDA public API.

## Features
- Search any drug by brand name
- View purpose, warnings and adverse reactions
- Recent search history
- Copy generic name to clipboard
- Responsive design (mobile-friendly)

## Technologies
HTML · CSS · JavaScript · OpenFDA API
